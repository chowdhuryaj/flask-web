## Decisions Made

- **New repo, not a subdir of AdeptCompanion** — clean Pages hosting, own history. (User chose.)
- **Vanilla JS, zero build** — AlooMapper (`~/hid-remapper/config-tool-vial`) pattern; push = deploy. No framework, no npm deps except one vendored file.
- **Vendored `xz-decompress` 0.2.3 as a CLASSIC `<script>`** (not ESM) — its UMD wrapper needs a real top-level `this`, which ES modules null out. Exposes `window['xz-decompress'].XzReadableStream`. Verified it decodes Python `lzma.compress` output (the plan's one true unknown — passed first try).
- **HUD = Document Picture-in-Picture**, in-page draggable corner-snap overlay as fallback. `styles.css` is deliberately the ONLY stylesheet so the PiP clone is a single `<link>`.
- **`hid.pause()` is advisory only** — it gates the HUD poll tick, it must NOT block the request queue. (First draft blocked the queue in `_send`; that deadlocked the unlock transaction which has to run *while* paused. Fixed.)
- **Clamp-echo everywhere** — every setter adopts the value the firmware echoes back, never its own. Mirrored from the Swift app's hard-won rule.

## Post-v1 Porting Roadmap (codecs already landed where noted)

Order per the approved plan. Each ports a Swift view; the protocol codecs it needs mostly already exist in `vialproto.js`/`vialclient.js`.

1. [ ] **Macros** — new `macros-tab.js`. Codec DONE (`vialproto.js MacroCodec`, `vialclient.js macro*`). Source: `AdeptCompanion/Sources/AdeptCompanion/MacrosView.swift`. Gotcha: `writeMacroBuffer` is UNLOCK-GATED (silently ignored while locked) — check unlock, write, re-read to verify.
2. [ ] **Tap dance** — codec DONE (`TapDance`, `tapDanceGet/Set`). Source: `TapDanceComboViews.swift`. Slot keycodes may be QK_KB customs or MO(layer). This fork's Vial owns `tap_dance_actions[]` — dynamic EEPROM entries only (firmware note in vial-qmk CLAUDE.md).
3. [ ] **Combos** — codec DONE (`Combo`, `comboGet/Set`) + per-combo layer masks channel `0x20` (`CH.comboLayers`, `slot.comboMask`, caps `comboLayerMasks`). Source: `TapDanceComboViews.swift`. 4 inputs + output, positional.
4. [ ] **Key overrides** — codec DONE (`KeyOverride`, 32 slots). Source: `KeyOverrideView.swift`. `defaultOptions = 0b10000111`; mod masks are 8-bit real-mod format.
5. [ ] **RGB painter (NLKB16)** — channel `0x21`, caps `rgbMap`. **PAYLOAD-ADDRESSED** frames (`getBytes/setBytes`, never u16): GET led `[layer,led]→[layer,led,H,S,V]`; SET led `[layer,led,H,S,V]`; bulk `[layer,start,count≤8,count×HSV]`; fill `[layer,H,S,V]`. HSV 0–255. Plus VialRGB effect engine (`data[1] ∈ 0x40–0x44`, 2-byte header → `rawCommand`, NOT the Flask shape). Sources: `NLKB16Model.swift`, `TileGridTabs.swift`; VialRGB consts in `AdeptProtocol.swift` bottom.
6. [ ] **Display tab (NLKB16)** — channel `0x22`, caps `display`/`displayWidgets`/`bigDisplay`. Push `[0x10, logical line, ASCII…]`, release `0x11`, widgets `slot.dispWidget(line)` ids from `NLKB.widgetNames`, custom text `slot.dispCustom(line)`, sleep `0x0A`, overlays `0x0B`. **Glass = 64×32 window; only logical lines 4–11 × cols 0–4 visible** (v5: 4 double-height big lines 0–3). Sources: `NLKB16Model.swift`, `NLKB16Tabs.swift`, `TileGridTabs.swift`.
7. [ ] **Gestures (Adept/Sval)** — channel `0x11`, caps `gestures`. 8 sets × 8 dirs, `slot.gesture(set,dir)` (internal order E SE S SW W NW N NE; cardinals 0x10 block, diagonals 0x30 block). Active-set 0x02, ratchet 0x01. Slots fire raw keycodes via tap_code16 — basic + C()/S()/A()/G() only, no macros/layer/QK_KB. Source: `GesturesTab.swift`, `Keycodes.swift`.
8. [ ] **Wheel chords** — channel `0x1C`, caps `wheelChords`. 8 buttons × 8 dirs `slot.wheelChord`, enabled 0x01, step 0x02, holdMs 0x03. Source: `MouseChordsTab.swift`.
9. [ ] **.vil save/load** — Vial-GUI-compatible schema (int keycodes so Vial GUI opens Flask files) + Flask extension keys `flask_tunings` + `flask_rgbmap`. Source: `AdeptCompanion/Sources/AdeptCompanion/LayoutFile.swift`. This is the bootloader-erase restore path for NLKB16.
10. [ ] (lower priority) Typing test, Tap calibrator — self-contained UI, no new protocol. Sources: `TypingTestTab.swift`, `TapCalibratorTab.swift`.
